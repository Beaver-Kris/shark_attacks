# Shark Attack Analysis

**Data cleaning · SQL analysis · Power BI dashboard**

Analysis of global shark attack data to identify trends, seasonality, and risk factors.

---

## 📊 Key Insights

- Fatality rate: **~25%** 
- Most attacks: **USA** and **Australia**
- Most dangerous by amount of attacks: **Great White Shark**
- Most dangerous by percent of fatal outcome: **Tiger Shark**

---

## 🛠 Tools

- **Python (Pandas)** – data cleaning
- **PostgreSQL** – database
- **SQL** – window functions, CTEs, aggregations
- **Power BI** – dashboard

## 🚀 Quick Start

1. Run `notebooks/data_cleaning.ipynb`
2. Import cleaned CSV to PostgreSQL
3. Run SQL queries from `/sql`
4. Open Power BI dashboard from `/dashboard`
