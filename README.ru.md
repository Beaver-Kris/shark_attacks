# Анализ нападений акул

**Очистка данных · SQL-анализ · Дашборд в Power BI**

Анализ данных о нападениях акул по всему миру: тренды, сезонность, факторы риска.

---

## 📊 Основные выводы

- Смертность: **~25%** (очень низкая)
- Больше всего нападений: **США** и **Австралия**
- Самый опасный вид по количеству нападений: **Большая белая акула**
- Самый опасный вид по проценту смертельность: **Тигровая акула**

---

## 🛠 Инструменты

- **Python (Pandas)** – очистка данных
- **PostgreSQL** – база данных
- **SQL** – оконные функции, CTE, агрегации
- **Power BI** – дашборд

---

## 📁 Структура проекта

shark_attacks/
├── data/ # сырые и очищенные данные
├── notebooks/ # ноутбук с очисткой
├── sql/ # все SQL-запросы
├── dashboard/ # файл .pbix и pdf
└── README.ru.md


---

## 🚀 Как запустить

1. Запустить `notebooks/data_cleaning.ipynb`
2. Загрузить очищенный CSV в PostgreSQL
3. Выполнить SQL-запросы из папки `/sql`
4. Открыть дашборд из папки `/dashboard`
